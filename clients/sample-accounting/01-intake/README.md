# Sample Client Intake Placeholder

This directory is intentionally public-safe.

Private discovery notes, stakeholder details, company profiles, and production
volumes were removed before publication. Use this folder only for synthetic
sample intake material that can be shared publicly.

## Example scope

- Client: Example Accounting Studio
- Contact: operations@example.com
- Data sources: sample inbox and sample document storage
- Objective: demonstrate an invoice-processing discovery workflow without
  exposing private client information.
