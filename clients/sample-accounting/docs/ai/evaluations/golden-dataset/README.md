# Synthetic Evaluation Fixtures

This directory intentionally contains no real invoice fixtures.

Any future fixture added here must be generated from fictional companies,
synthetic tax identifiers, and `example.com` contact data. Reverse mappings to
real documents are prohibited.

## Fixture rules

- Use fictional supplier and customer names.
- Use synthetic identifiers such as `PT000000001`.
- Use generated PDFs or text files only.
- Never include production invoices, private emails, or reverse-anonymization
  notes.
